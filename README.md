# Jiayi Xie's Academic Homepage

This is a customized Academic Pages site for `GAry45v.github.io`.

It is based on the [academicpages/academicpages.github.io](https://github.com/academicpages/academicpages.github.io) template and is structured as a concise academic homepage.

## Main Sections

- `Home`: concise intro, publications, education, internships, honors, and skills
- `CV`: compact academic and engineering profile
- `Blog Posts`: technical notes and long-form essays

## Files Customized

- `_config.yml`: site title, URL, author sidebar, theme, repository
- `_data/navigation.yml`: top navigation
- `_pages/about.md`: concise homepage
- `_pages/cv.html`: redirect to the public CV PDF
- `_pages/year-archive.html`: blog archive
- `_publications/`: publication entries
- `_portfolio/`: project entries
- `_posts/`: writing entries

## Public Information

- The profile photo is stored at `images/self.jpg`.
- Contact identifiers are intentionally presented as label-only copy actions on the site, but remain visible in the public repository source.
- Publication status wording should be updated whenever decisions change.

## Publish on GitHub Pages

Create a public repository named:

```text
GAry45v.github.io
```

Then push this folder to the repository's `main` branch. GitHub Pages will publish it at:

```text
https://GAry45v.github.io
```

The Academic Pages template recommends editing `_config.yml` and setting `url` and `repository` to the final GitHub Pages location. This has already been configured for `GAry45v/GAry45v.github.io`.
